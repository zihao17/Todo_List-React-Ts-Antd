# Todo List 任务清单应用

[English](#english) | [中文](#chinese)

## English

A modern Todo List application built with React, TypeScript and Ant Design.

### Features

- Task management (add/edit/delete)
- Task status tracking
- Recycle bin functionality
- Responsive design
- Statistics dashboard
- Motivational quotes

### Tech Stack

- React
- TypeScript
- Ant Design
- Vite
